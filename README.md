# 🚀 Автоматизация и ML: Связка FastAPI & n8n

Этот проект представляет собой готовую среду в Docker, объединяющую Python-бэкенд на FastAPI и систему автоматизации сценариев n8n. Он отлично подходит для машинного обучения, выполнения Python-кода через API и построения автоматизированных пайплайнов.

---

## ⚡ Как запустить проект (Быстрый старт)

### Шаг 1. Установите Docker
Если у вас ещё не установлен Docker, скачайте и установите **[Docker Desktop](https://www.docker.com/products/docker-desktop/)**.
> ⚠️ Убедитесь, что программа Docker Desktop запущена на вашем компьютере перед тем, как переходить к Шагу 2.

### Шаг 2. Запуск контейнеров
Откройте терминал (командную строку) в папке с этим проектом и выполните команду:
```bash
docker-compose up --build
```
*Эта команда скачает нужные компоненты, соберет проект и запустит серверы. Дождитесь, пока логи в терминале перестанут активно бежать.*

---

## 🌐 Доступ к сервисам

После успешного запуска вам будут доступны два главных сервиса:

### 1. 🔗 n8n (Визуальное создание сценариев/Workflows)
Система автоматизации доступна в браузере:
- **Ссылка:** [http://localhost:5678](http://localhost:5678)
### 2. 🔧 Бэкенд (FastAPI)
Python-сервер, который обрабатывает данные и запускает код:
- **Доступ в браузере (для просмотра работоспособности):** [http://localhost:8000](http://localhost:8000)
- **Внутренний адрес (используйте его внутри нод n8n при запросах):** `http://fastapi:8000`

#### 📡 Основные API Эндпоинты FastAPI
- **`GET /health`** — Проверка, что сервис работает (статус-чек).
- **`POST /execute`** — Безопасное выполнение переданного Python-кода прямо в среде бэкенда.
- **`GET /data/files`** — Возвращает список всех датасетов, которые лежат в общей папке `/app/data`.
- **`GET /check-imports`** — Проверка установленных версий библиотек (numpy, sklearn, и др.).

## 💾 Сохранение рабочих процессов (Workflows) в n8n

Сценарии в n8n нужно сохранять вручную, чтобы не потерять их при перезапуске проекта:
1. В интерфейсе n8n нажмите **Publish**, чтобы сохранить текущий пайплайн.
2. В верхнем меню выберите **Export as JSON** → **Download as JSON**.
3. Закиньте скачанный файл в папку вашего проекта по пути: `./n8n/workflows/`.

## 📂 Из чего состоит проект

```text
📁 Корень проекта
├── 📁 data/                 # Папка с данными и датасетами (например, Iris.csv). Доступна всем сервисам.
├── 📁 fastapi/              # Исходный код Python-сервера (FastAPI)
│   ├── main.py              # Логика работы всех эндпоинтов сервера
│   └── Dockerfile           # Инструкция для сборки образа Python-бэкенда
├── 📁 models/               # Папка для сохранения обученных ML-моделей 
├── 📁 n8n/workflows/        # Папка для сохранения бэкапов ваших n8n-сценариев (.json)
├── docker-compose.yml       # Файл конфигурации для связки и запуска n8n и FastAPI
└── README.md                # Этот файл документации
```
